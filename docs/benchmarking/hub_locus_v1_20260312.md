## Analysis & Insights

### 1. Pose Accuracy Leap
- **Origin Alignment:** Previous benchmarks reported ~9cm P50 translation errors. Correcting the alignment between the detector's Top-Left origin and the generator's Center-based ground truth has revealed the true accuracy of the pipeline, with P50 errors now in the **5mm to 15mm** range.
- **Rotation stability:** Median rotation error remains excellent, staying below 5° even at 1080p.

### 2. Detection Performance
- **Recall:** 100% recall is maintained at 480p and 720p. At 1080p, a single extremely distorted tag was missed (97.78% recall), representing the current edge of the detector's capability.
- **RMSE:** Subpixel accuracy is consistently around 1.0-1.4 px RMSE, demonstrating the effectiveness of the Erf-based refinement across scales.

### 3. Pipeline Latency
- **Efficiency:** In release mode, processing latencies range from **~18ms (480p)** to **~37ms (1080p)**. This confirms that the pipeline is capable of real-time performance on standard hardware.

## Methodology
- **Data Source:** Hugging Face `single_tag_locus_v1_tag36h11` subsets.
- **Harness:** `locus-core` regression suite with Erf-based 1-parameter subpixel solver.
- **Environment:** Linux, Release Profile.
